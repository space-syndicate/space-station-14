Space Station 14 это ремейк SS13, который работает на собственном движке [Robust Toolbox](https://github.com/space-wizards/RobustToolbox), написанном на C#.

Это репозиторий первого русскоязычного сервера по Космическим Сражениям в Space Station 14.
## Приоритеты сервера
1. Свобода - сервер распространяется по принудительно свободной лицензии APGLv3 для поддержания комьюнити сс14.
2. Новые механики - красивая и оптимизированная система боев и должностей.
3. Слаженный коллектив - работа над сервером учитывает желания команды и игроков проекта.
4. Закон - следование правилам и идеям независимого сообщества сс14.

## Сборка
1. Склонируйте этот репозиторий локально
2. Запустите `RUN_THIS.py` для инициализации подмодулей и скачивания движка.
3. Скомпилируйте проект.

[Более подробная инструкция по запуску проекта.](https://docs.spacestation14.com/en/general-development/setup.html)
