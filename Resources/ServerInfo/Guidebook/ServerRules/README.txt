These files contain Wizard's Den server rules. Since they reference Wizard's Den, they should not be used
by other servers without at least enough modification to not mislead players into thinking that they are
playing on Wizard's Den.

The filenames used for the rules files are not themselves rules. Only the contents of the files are rules.

==========================================================================================================

Эти файлы содержат правила сервера Wizard's Den. Поскольку они отсылают к Логову Волшебника, их не следует использовать.
другими серверами, по крайней мере, без достаточной модификации, чтобы не вводить игроков в заблуждение, заставляя их думать, что они
играю в Волшебном Логове.

Имена файлов, используемые для файлов правил, сами по себе не являются правилами. Правилами является только содержимое файлов.
